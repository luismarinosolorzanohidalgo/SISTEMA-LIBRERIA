# 📚 SISTEMA-LIBRERIA

Sistema de gestión para librerías de útiles escolares desarrollado en **PHP y MySQL**, diseñado para administrar productos, ventas y control de inventario. Permite organizar los artículos disponibles, registrar ventas y mantener un seguimiento actualizado del stock dentro del negocio.

---

## 🛠 Funcionalidades principales

- ✅ Registro y gestión de productos escolares
- ✅ Control de inventario de útiles y artículos de oficina
- ✅ Registro de ventas
- ✅ Consulta de disponibilidad de productos
- ✅ Organización del catálogo de artículos
- ✅ Reportes básicos de ventas y stock

---

## 💻 Tecnologías utilizadas

- PHP  
- MySQL  
- HTML  
- CSS  
- JavaScript
